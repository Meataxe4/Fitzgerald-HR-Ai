Code review notes

  • Design system is solid. The site already exposes a clean token set (--navy, --amber, --rule, --w60) — these templates reuse them verbatim so anything you tweak on the site flows through.
  • Demo section is the strongest hook. The phone-mock chat with typed answers + auto-cycling scenarios is the single best ad format you have. post-fitz-in-action.html turns it into a static frame; ideal as carousel slide 1.
  • Slot-machine hook ("your chef is a no-show…") is the strongest single line on the homepage. ad-square-no-show.html leans on it for the paid ad.
  • Stats band ($12k saved, <90s, 24/7, 100% AU law) — high-converting "by the numbers" social tile.
  • Watch-outs: index.html is 1,690 lines with all CSS + JS inline. Fine for a single-page marketing site, but the demo animation logic (lines 1526–1686) would be easier to maintain in js/.

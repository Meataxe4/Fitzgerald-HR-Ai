How to use these in Canva

  1. Editable vectors: In Canva, create a 1080 × 1920 px design, then Uploads → upload each .svg. Canva imports SVGs as editable elements (text, colours and shapes stay editable; webfonts remap to the closest Canva font — set Playfair Display, Outfit and DM Mono from Canva's font picker to match exactly).
  2. Quick raster route: open any card here in a browser and screenshot at 100%, or run the PNG export below, then upload the PNGs to Canva as full-bleed backgrounds.
  3. Assemble the Reel: drop the cards in order 0 → 6 on a Canva Reel/Video timeline (~6s each), add a trending audio track and burned-in captions, then export 1080×1920 MP4.

Optional: render PNGs locally

From marketing/reel-cards/, with rsvg-convert (librsvg) or Inkscape installed:
  1. for f in card-*.svg; do rsvg-convert -w 1080 -h 1920 "$f" -o "${f%.svg}.png"; done
  2. or inkscape card-0-hook.svg --export-type=png -w 1080 -h 1920
